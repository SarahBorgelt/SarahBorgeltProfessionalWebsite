# Sarah Borgelt | Personal Portfolio

A personal portfolio website showcasing my experience, projects, technical skills, professional background, and community involvement.

The website is designed to provide an accessible, responsive, and user-friendly experience while highlighting my work as a Software Engineer and Data Engineer.

## 📌 Overview

This portfolio was built from the ground up to showcase my professional experience, software development projects, technical skills, and personal interests.

The site focuses on:

* Responsive design across desktop and mobile devices
* Clean and maintainable React architecture
* Accessibility best practices
* Search engine optimization (SEO)
* Modern frontend development practices

---

## 🚀 Features

* Multi-page React application
* Client-side routing with React Router
* Dynamic page metadata using React Helmet Async
* SEO optimization with:

  * Custom page titles
  * Meta descriptions
  * Open Graph tags
  * Canonical URLs
  * Structured data (JSON-LD)
* Responsive navigation
* Custom 404 page handling
* Accessible UI components
* Contact form integration
* Optimized assets and modern web practices

## 💻 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/SarahBorgelt/<repository-name>.git
```

Navigate into the project:

```bash
cd <repository-name>
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will be available locally through the Vite development server.
